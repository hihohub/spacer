Ztree

To install a datatree on your web page, simply include a container element with the id DATATREE and an indented outline between the tags, here a poem by Lewis Caroll. Or, import the indented text file with the attribute src='filename'. Ztree also accepts an HTML list, but you will have to add the attribute type='html'. The 'root' attribute designates the text for the root node, here 'a poem'. The 'toolbar' attribute adds buttons, here 'collapse,expand'. The 'treeheight' attribute restricts the height. We have also added a style attribute of width:50%;margin-left:25%. Finally, import Ztree10.0.js and add another script with one line in the onload handler, ZTREE.AutoInit();

Other toolbar attributes of interest (for larger data trees): search, search_horizontal, search_vertical, next, previous, reset, number, replace, edit, save, load, alphabetize. If the toolbar needs scrollbars or other styling, it should be possible to style it with CSS with the id datatree_toolbar.

Other attributes of interest: lettering='font', open, closed, and empty for the three types of arrows. The arrows may be replaced with text, tags, or images. For example, open='+', closed='-', empty='#', or the same text between span tags (within quotes). Or, html img tags (between quotes) for each of them with images of your choice.
