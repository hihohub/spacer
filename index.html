SPACER

To install a datatree on your web page, simply include a container element with the id DATATREE and an indented outline between the tags. Or, import the indented text file with the attribute src='filename'. Spacer also accepts an HTML list, but you will have to add the attribute type='html'. The 'root' attribute designates the text for the root node, here 'Example'. The 'toolbar' attribute adds buttons, here 'collapse,expand,search,reset'. The 'treeheight' attribute restricts the height. We have also added a style attribute of width:50%;margin-left:25%. Finally, import Spacer10.1.js and add another script with one line in the onload handler, SPACER.AutoInit();

Toolbar attributes of interest

search, search_horizontal, search_vertical, next, previous, reset, number, replace, edit, save, load, html, text, and alphabetize. If you add an 'edit' button, you'll also need 'save' and 'load'. The 'save' button has been changed to the same as the 'html' button, which saves an html list. Once text or html has been loaded into the tree, it's converted to html, so it's easier to save it as html. If you prefer plain text, the 'text' button saves just the text, however it's not working well, and typically only works once. As for styling, Spacer was designed for full-screen trees, and may sometimes not style properly for small trees. For example, excessive toolbar buttons will wrap to the next line on a small tree. If the toolbar needs scrollbars or other styling, it may be possible to style it with CSS with the id spacer_toolbar.

Attributes of interest

Attributes are set within the container tag. Other attributes are lettering='font' for the font family, open, closed, and empty for the three types of arrows. The arrows may be replaced with text, tags, or images. For example, open='+', closed='-', empty='#', or the same text between span tags (within quotes). Or, html img tags (between quotes) for each of them with images of your choice.

Properties of interest

Properties are set from javascript. After AutoInit, set properties of the SPACER.TREE object. If you have multiple trees, they are stored in the TREES array, so set properties of, for example, SPACER.TREES[1], starting from zero. Some helpful properties of the tree object are HIGHLIGHT_BACKGROUND_COLOR, HIGHLIGHT_TEXT_COLOR, and TYPE. The TYPE is either "text" or "html", depending on the file extension of the file you loaded into the tree. The default is "text". If you want to load an html list into the tree, you must change the type to "html" first. After AutoInit, write SPACER.TREE.TYPE = "html".

Multiple trees on same page

The default id is DATATREE. Instead, give each tree a different id attribute, then initialize each of them with their id as a parameter. For example, SPACER.AutoInit('tree1'); SPACER.AutoInit('tree2'); SPACER.AutoInit('tree3');

Next version

This repository was written with an ancient version of javascript. The correct implementation should use a style similar to that in github.com/hihohub/elevator. A new version has been started but will take a long time to complete. If you try to install this outdated version on your website, please be aware that it does not yet play well with other HTML or javascript content on the same page. Also, due to the large code footprint, it's probably not possible to customize it except as specified in the instructions.
