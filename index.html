Spacer

The name 'Spacer' refers to the spaced text that serves as input to the tree. The repository has had several name changes recently. If you were using the repository with one of the previous names, we apologize.

To install a datatree on your web page, simply include a container element with the id DATATREE and an indented outline between the tags. Or, import the indented text file with the attribute src='filename'. Spacer also accepts an HTML list, but you will have to add the attribute type='html'. The 'root' attribute designates the text for the root node, here 'Example'. The 'toolbar' attribute adds buttons, here 'collapse,expand'. The 'treeheight' attribute restricts the height. We have also added a style attribute of width:50%;margin-left:25%. Finally, import Spacer10.0.js and add another script with one line in the onload handler, SPACER.AutoInit();

Other toolbar attributes of interest

search, search_horizontal, search_vertical, next, previous, reset, number, replace, edit, save, load, alphabetize. If the toolbar needs scrollbars or other styling, it should be possible to style it with CSS with the id spacer_toolbar.

Other attributes of interest

Other attributes of interest: lettering='font', open, closed, and empty for the three types of arrows. The arrows may be replaced with text, tags, or images. For example, open='+', closed='-', empty='#', or the same text between span tags (within quotes). Or, html img tags (between quotes) for each of them with images of your choice.

Multiple trees on same page

The default id is DATATREE. Instead, give each tree a different id attribute, then initialize each of them with their id as a parameter. For example, SPACER.AutoInit('tree1'); SPACER.AutoInit('tree2'); SPACER.AutoInit('tree3');
