SPACER

To install a datatree on your web page, simply include a container element with an id, for example "DATATREE", and an indented outline between the tags. Or, import the indented text file with the attribute src='filename'. Spacer also accepts an HTML list, but you will have to add the attribute type='html'. The 'root' attribute designates the text for the root node, here 'Example'. The 'toolbar' attribute adds buttons, here 'collapse,expand,search,reset'. The 'treeheight' attribute restricts the height. We have also added a style attribute of width:50%;margin-left:25%. Finally, import Spacer10.2.js and add another script with two lines, let SPACER = new Spacer();onload = () => {SPACER.AutoInit("DATATREE");}

Toolbar attributes of interest

search, search_horizontal, search_vertical, next, previous, reset, number, replace, edit, save, load, html, text, alphabetize, and publish. If you add an 'edit' button, you'll also need 'save' and 'load'. The 'save' button has been changed to the same as the 'html' button, which saves an html list. To save text, add a 'text' button. To publish a tree file, add a 'publish' button, then copy the text into an html file.

Attributes of interest

Attributes are set within the container tag. The attributes 'src' and 'type' allow you to load a file. If the file is a text file, set type='text'. If the file is an html list, set type='html'. Other attributes are lettering='font' for the font family, open, closed, and empty for the three types of arrows. The arrows may be replaced with text, tags, or images. For example, open='+', closed='-', empty='#', or the same text between span tags (within quotes). Or, html img tags (between quotes) for each of them with images of your choice.

Properties of interest

Properties are set from javascript. After AutoInit, set properties of the SPACER.TREE object. If you have multiple trees, they are stored in the TREES array, so set properties of, for example, SPACER.TREES[1], starting from zero. Some helpful properties of the tree object are HIGHLIGHT_BACKGROUND_COLOR, and HIGHLIGHT_TEXT_COLOR.

Multiple trees on same page

The default id is DATATREE. Instead, give each tree a different id attribute, then initialize each of them with their id as a parameter. For example, SPACER.AutoInit('tree1'); SPACER.AutoInit('tree2'); SPACER.AutoInit('tree3');
